Modelo de caja(Box Model)

Es un concepto fundamental en el desarrollo web que describe cómo se estructuran los elementos en una página web y cómo se calculan sus dimensiones.
  • Contenido: Es el área real que contiene el texto, imágenes u otros elementos.
  • Relleno(Padding): Espacio transparente alrededor del contenido. No tiene un color de fondo propio. Le afecta el color de fondo del elemento
  • Borde(Border): Es un borde alrededor del relleno. El borde puede tener un color propio o verse afectado por el color de fondo del elemento. El borde tiene varias propiedades
  • Margen(Margin): Espacio transparente fuera del borde. No tiene un color de fondo propio, ni le afecta el color de fondo del elemento. Se utiliza para crear espacio entre elementos.
